Learn more about Git

Here are some basic commands that you should commit to memory when learning Git:

  • git clone: copies an existing repository (project) to your computer.
  • git pull: pulls changes made to the repo from other contributors onto your computer.
  • git status: checks the status of the repo, alerting you to any changes made.
  • git add: adds your changes to files to be submitted to the repo.
  • git commit: commits (stages, semi-permanently adds) your changes to the repo, with a message.
  • git push: pushes (permanently adds) your changes to the repo for others to see.
